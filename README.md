# Bibi-Release
这里是某个B开头的UWP app的appx分发中心，本体已经由于大人的原因无法继续在微软商店中公开发布了。

最低系统版本要求为16299（ARM版为15063）。

# 安装方法
1、从Releases中下载对应系统平台（目前支持X64和ARM）的appx。

2、下载Atelier39.cer，打开并安装证书（本地计算机->证书存储->选择“受信任人”("Trusted People")）。

3、打开appx并安装。
